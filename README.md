# 🇸🇬 新加坡留学打包清单

一个纯前端的行李打包清单工具。

最初是为了准备去新加坡留学时整理行李而写，后来逐渐加入了拖拽装箱、搜索定位、本地保存、导入导出等功能。所有数据都保存在浏览器本地，不需要登录，也不依赖任何后端。

🌐 在线体验（GitHub Pages）  
https://a1-nair.github.io/sg-packing-list/

---

## 功能

- 📦 将物品拖拽到不同箱子进行整理
- ✅ 勾选已打包物品，实时统计整体进度
- 🔍 搜索物品并快速定位
- 📝 为每件物品添加备注
- 📂 自定义分类、物品和箱子
- 💾 使用 LocalStorage 自动保存数据
- 📤 导出 / 📥 导入 JSON 备份

---

## 使用方法

1. 新增自己的分类、物品和箱子，或直接使用默认清单。
2. 将物品拖入对应箱子，记录实际放置位置。
3. 打包完成后勾选对应物品，随时查看整体进度。
4. 建议定期导出 JSON 文件作为备份。

## 为什么做这个项目？

这个项目其实没有什么宏大的目标。

一开始只是因为我准备去新加坡留学，面对一长串 Excel 清单时觉得不够直观，想着：「既然 HTML 更方便，那为什么不自己写一个？」于是就在 AI 助手（感谢 ChatGPT 和 Gemini）的帮助下，一点一点把它做了出来。

等到真正打包的时候，我才发现市面上其实已经有不少类似的工具，只是它们各自有不同的侧重点：有些功能很多，但上手成本比较高；有些更适合旅行，而不是留学这种需要整理大量物品的场景。

这个项目主要还是为了满足自己的需求，所以默认清单会更偏向留学。不过所有内容都可以自由修改，也许搬家、旅行或者其他需要整理物品的时候也能派上用场。

我的代码水平并不算高。如果自己使用过程中发现了 Bug，我会继续慢慢修；如果有人提出建议、发现问题，或者愿意帮忙改进，那就再好不过了。

如果这个小工具也刚好能帮到你，希望它能让打包这件事轻松一点。
