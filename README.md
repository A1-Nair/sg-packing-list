# 🇸🇬 新加坡留学打包清单 (Singapore Packing List)

> 一个轻量、丝滑、纯前端的行李打包清单管理工具，专为赴新加坡留学的你打造。告别杂乱无章的备忘录，用拖拽和进度条治愈你的打包焦虑！

🌍 **在线体验 / Live Demo**：[https://a1-nair.github.io/sg-packing-list/](https://a1-nair.github.io/sg-packing-list/)

---

## ✨ 核心功能 (Features)

| 功能模块 | 亮点描述 |
| :--- | :--- |
| 📦 **全景与装箱双模式** | 左侧总览所有分类与物品状态，右侧吸顶悬浮箱子面板，支持直观的拖拽装箱。 |
| 🔍 **精准搜索与高亮** | 输入关键词一键定位，自动展开折叠的大类并红框高亮提示，搜完一键恢复全景视图。 |
| 📊 **实时进度跟踪** | 顶部的总进度条与右侧每个独立箱子的 `(已装/总数)` 进度显示完美结合。 |
| 💾 **本地安全存储** | 数据完全保存在浏览器的 LocalStorage 中，隐私绝对安全，断网可用，刷新不丢失。 |
| 🔄 **数据导入与导出** | 支持一键导出 JSON 备份文件，更换电脑或浏览器也能无缝恢复打包进度。 |

## 🚀 如何使用 (Usage)

1. **建立清单**：点击 `新增物品`、`新增分类` 或 `新增箱子` 建立你的专属结构（系统已默认自带一套涵盖证件、电子、衣物、药品的留学基础清单）。
2. **整理装箱**：鼠标长按左侧的物品卡片，直接拖拽到右侧对应的箱子中。或者点击卡片上的 `📦 移出箱子` 重新分配。
3. **状态标记**：打包好一件勾选一件，也可以通过 `＋/－` 按钮精确调整物品数量。
4. **分类折叠**：点击左侧大类标题，可自由折叠/展开该分类，保持界面清爽。
5. **备份数据**：由于数据储存在本地浏览器中，强烈建议在完成阶段性打包后，点击 `📤 导出清单` 将进度下载到本地，以防浏览器缓存被意外清空。

## 🛠️ 技术细节 (Tech Notes)

- **技术栈**：纯 HTML / CSS / Vanilla JavaScript。
- **架构**：零外部依赖，单文件（Single File）运行，极速加载。
- **自适应**：采用 CSS Grid 与 Flexbox 布局，右侧面板支持 Sticky 吸顶悬浮，解决长列表滚动痛点。
- **部署**：基于 GitHub Pages 自动化部署，主分支 `main` 更新即自动上线。
